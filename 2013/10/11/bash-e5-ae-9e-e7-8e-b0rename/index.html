bash实现rename
Post：2013-10-11 12:45:58 Tags：/ python / bash / linux / egg / import / rename / this / Visit：

昨天被问道个重命名文件的问题要求bash实现，一时没有想起怎么实现只想到个rename命令也由于没有开发环境不能调试就放弃了。回家后想了想可以如下完成。 一 关键是${file/.png}

#!/bin/bash
lists=\`ls *png\`
for file in $lists;do
    mv "$file" "${file/.png}".jpg; 
done

二 关键是basename


#!/bin/bash
lists=\`ls *png\`
for file in $lists;do
        mv $file \`basename $file .png\`.jpg;
done
