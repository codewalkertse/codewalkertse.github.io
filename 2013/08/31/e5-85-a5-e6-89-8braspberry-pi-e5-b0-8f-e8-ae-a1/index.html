入手raspberry pi小计
Post：2013-08-31 20:44:20 Tags：/ linux / RaspberryPi / Visit：

update 2013-09-20 前天最后一天上班，中午去去了西数的2T绿盘+盒子，我的nas必要的部件终于凑齐了。晚上话时间把所有文件全部迁移到了此盘中。但有个奇怪的问题是我笔记本只有一个usb口认识硬盘盒，但是Rasperry Pi就完全没问题，随便热插拔。这也是应为盒子是自带电源的缘故吧大概. ################### 今天入手了raspberry pi。 淘宝上买的，第一个结果就是本地的老板是个年轻小伙，于是愉快的入了。 这家店入的清单如下：机子,散热片2,电源(也就是变压器),送了电源线以及塑料盒子。 很早钱在电脑城去入的.SD卡(8G)，HDMI线. 1下载镜像 这里 ,按照自己喜好下载系统。我还是选择了我喜爱的arch，不过这是ARM版本，其实在感受一下午之后暂时没觉得*用法有什么不同。 烧录镜像到SD卡，方法有二 2.1 win： win32diskimager 可以写入。这是网上推荐的软件，同时每次我烧录各种镜像到U盘安装系统的时候也是用的他。 2.2 Linux： dd bs=1M if=2012-12-16-wheezy-raspbian.img of=/dev/sdc** 这里需要注意最后的/dev/sdc 要按照自己的信息来写，因人而异。我自己机子上本来就挂了两块硬盘，所以序列号自动就变成了sdc. 2.3 显示问题 使用HDMI连接显示器会没有信号，原因是默认没有走这个接口输出视频信号(猜测).需要修改配置文件这一步可以在烧录镜像之后立即执行 在SD卡的根目录下找到config.txt 在最后加入如下设置 hdmi_ignore_edid=0xa5000080 hdmi_force_hotplug=1 hdmi_group=2 hdmi_mode=4 hdmi_drive=2 config_hdmi_boost=4 sdtv_mode=2 arm_freq=800 hdmi_mode=4 是你显示器的分辨率，可以参照http://www.shumeipai.net/forum.php?mod=viewthread&tid=19317 这里按照自己的所想修改. 3找回“丢失”的空间 不管多大的SD卡安装之后系统就只能找到2G左右的空间,找回很简单.由于我操作的时候没有截图所以就不能上图了. 3.1进入系统之后fdisk,先查看分区表就会发现还有空间未使用 3.2新建逻辑分区 3.3写入分区表，这个时候可能会报错，说xxx写入失败，下次重启才会生效. 3.4重启之后就可以看到新分区了,由于是SD卡中所以名字应该类似 /dev/mmcblk0pX 这种而不是常见的sdX 或者hdX. 4设置语言 vim /etc/locale.gen 把英文和中文的都反注释掉 locale-gen 使之生效 vim /etc/locale.conf 写入一下内容 LANG=en_US.UTF-8 LC_CTYPE=”en_US.UTF-8” LC_NUMERIC=”en_US.UTF-8” LC_TIME=”en_US.UTF-8” LC_COLLATE=”en_US.UTF-8” LC_MONETARY=”en_US.UTF-8” LC_MESSAGES=”en_US.UTF-8” LC_PAPER=”en_US.UTF-8” LC_NAME=”en_US.UTF-8” LC_ADDRESS=”en_US.UTF-8” LC_TELEPHONE=”en_US.UTF-8” LC_MEASUREMENT=”en_US.UTF-8” LC_IDENTIFICATION=”en_US.UTF-8” LC_ALL= 5关于时间 ln -s /usr/share/zoneinfo/Asia/Chongqing /etc/localtime 设置时区。 由于Rsspberry Pi 没有硬件时钟所以最好还是用ntp来修正时间 vim /etc/ntp.conf 把server改成下面的内容 server 0.asia.pool.ntp.org server 1.asia.pool.ntp.org server 2.asia.pool.ntp.org server 3.asia.pool.ntp.org 差不多就这些吧，等过段时间入了大硬盘充实了NAS之后再写. 打完收工.
